## Mục tiêu
- CRUD Todo trên Postgres
- /health, /metrics hoạt động
- Traces xuất hiện trong Jaeger

## Các bước
1) `docker compose up --build`
2) Mở FE 8080, thêm/xóa todo
3) Kiểm tra /metrics, Jaeger UI

## Kết quả mong đợi
- Tạo/xóa todo OK
- Metrics có http_requests_total tăng
- Jaeger thấy spans cho các request

## Lỗi thường gặp
- Backend không kết nối DB → kiểm tra ENV DB_*
- Không thấy trace → kiểm tra OTEL_EXPORTER_OTLP_ENDPOINT và cổng 4318
- /metrics 404 → quên đăng ký register cho prom-client
